Porta San Donato (also known as Porta Zamboni)

Porta San Donato, also known as Porta Zamboni, was built in the 13th century on the road to Argenta and Ferrara. It was part of a bigger complex together with houses for guards. A drawbridge over the moat was built in 1354. The gate was closed and walled up for security reasons in 1428 but reopened again some decades later. As the gate San Donato is located on the crossing of main streets (the ring boulevard and Via San Donato), it run the risk to be demolished several times, especially in 1952 and 1959 when finally only one metre of wall was destroyed after a heated debate to resolve traffic problems. I decided to enrich this entity because it is one of the most iconic gates of Bologna, since it is located at the very beginning of Via Zamboni, which leads to the university area of the city

The following query searches for architectural heritage labeled as Porta San Donato in Bologna: 

PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> 
PREFIX arco: <https://w3id.org/arco/ontology/arco/> 
PREFIX arco-core: <https://w3id.org/arco/ontology/core/> 
PREFIX cis: <http://dati.beniculturali.it/cis/> 

SELECT DISTINCT ?porta ?label
WHERE {
  ?porta rdf:type arco:ArchitecturalOrLandscapeHeritage .
  { ?porta rdfs:label ?label }
  UNION { ?porta cis:name ?label }
  UNION { ?porta arco-core:hasTitle ?title .
           ?title rdfs:label ?label }
  UNION { ?porta arco-core:hasDenomination ?den .
           ?den rdfs:label ?label }
  FILTER (REGEX(STR(?label), "Porta San Donato", "i"))
  FILTER (REGEX(STR(?label), "Bologna", "i"))
}

Conclusion

Even though Porta San Donato (or Porta Zamboni) was already a described entity inside the ArCo ontology, thanks to LLMs, further research on other websites to verify the informations and SPARQL queries I was able to add some other very interesting informations, discovering myself at the same time new things about one of the main architectural heritage of Bologna. I considered worth focusing on both context description (for example the date of construction in the 13th century and the date of restoration in 2007-2009, the drawbridge added in 1354, the name in bolognese, the fact that Domenico Tibaldi was the designer of the gate based on the city defenses modernization plans) and denotative description (the materials used such as brick and sanstone; the achitectural style which is military renaissance architecture). At the beginning it wasn't easy because personally it's the first time that I work with LLMs to create queries, but overall It was a very interesting project to do, especially because we enriched the heritage of the city we're living in. To learn more about the challenges the group and I faced during this project, visit the Faced Challenges section of this website.
