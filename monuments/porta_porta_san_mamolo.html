Semantic Enrichment of Porta San Mamolo Using ArCo Ontology

Image: Porta San Mamolo (Poppi, Pietro, 1833‚Äì1914)

General Information

TypeArchitectural Heritage (Gate)
LocationBologna, Emilia-Romagna, Italy
Bolognese namePorta ed San Mamel
Construction12th century
Porta San Mamolo was one of Bologna‚Äôs city gates, historically significant for controlling the northern entrance. It was demolished in the early 20th century. Its profile has been enriched using ArCo ontology given its historical and architectural importance.

Verification of the data present on ArCo

Zero-shot SELECT query to check if Porta San Mamolo is in ArCo:

PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>

SELECT DISTINCT ?porta ?label
WHERE {
  ?porta rdf:type arco:ArchitecturalOrLandscapeHeritage .
  { ?porta rdfs:label ?label }
  UNION { ?porta arco-core:hasTitle ?title . ?title rdfs:label ?label }
  UNION { ?porta arco-core:hasDenomination ?den . ?den rdfs:label ?label }
  FILTER (REGEX(STR(?label), "Porta", "i"))
  FILTER (REGEX(STR(?label), "Bologna", "i"))
}
LIMIT 100

No results were found, confirming that Porta San Mamolo is not yet present in ArCo.

Explanation of relevant ArCo properties used

  • arco-cons:hasConstructionDevelopment / arco-cd:constructionPeriod: indicates the main construction period (XII century).
  • arco-den:hasMaterial: specifies the building material (brick).
  • arco-event:hasRelatedEvent: records the addition of the drawbridge in 1334.
  • arco-core:describes: stores descriptive information (e.g. dialect name).
  • arco-core:hasNote: records notes such as demolition date.

CONCLUSION

Technical Summary: Semantic Enrichment of Porta San Mamolo Using ArCo Ontology
This work presents the semantic modeling and RDF enrichment of historical data for Porta San Mamolo, a former city gate of Bologna, within the ArCo ontology framework.

Data Acquisition and Validation
Historical and architectural metadata were collected, including: construction in the 12th century, Bolognese dialect name ‚ÄúPorta ed San Mamel‚Äù, addition of a drawbridge in 1334, demolition in 1903, and building material brick.
SPARQL SELECT queries confirmed that Porta San Mamolo was absent from the ArCo knowledge graph (KG), validating the need for structured integration.

RDF Modeling via SPARQL CONSTRUCT
CONSTRUCT queries were generated to encode key properties:
Construction period ‚Üí arco-cons:hasConstructionDevelopment
Material ‚Üí arco-den:hasMaterial
Related event (drawbridge) ‚Üí arco-event:hasRelatedEvent
Descriptive information (dialect name) ‚Üí arco-core:describes
Demolition date ‚Üí arco-core:hasNote
Queries were expressed with canonical ArCo URIs and appropriate language tags (@en, @bo), producing RDF triples compatible with Virtuoso and other triple stores.

SPARQL Verification Queries
ASK queries were used to validate the presence of demolition information, returning false.
DESCRIBE queries confirmed the absence of Porta San Mamolo in the KG, demonstrating the need for insertion.

LLM-Assisted Query Generation
ChatGPT produced CONSTRUCT queries via one-shot prompting.
Gemini generated structured RDF triples using zero-shot prompting, using precise ArCo ontology terms.

Outcome
Porta San Mamolo‚Äôs historical, architectural, and descriptive data have been formally represented in RDF, ready for ingestion into the ArCo KG.
