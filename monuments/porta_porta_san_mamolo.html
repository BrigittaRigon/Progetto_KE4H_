Porta San Mamolo - Semantic Enrichment

Semantic Enrichment of Porta San Mamolo Using ArCo Ontology

Historical and Architectural Information

Construction date: XII century

Bolognese dialect name: Porta ed San Mamel

Drawbridge added: 1334

Demolition: 1903

Building material: bricks

Porta San Mamolo was one of Bologna‚Äôs city gates, historically significant for controlling the northern entrance. It was demolished in the early 20th century. Its profile has been enriched using ArCo ontology given its historical and architectural importance.

Verification of the data present on ArCo

Zero-shot SELECT query to check if Porta San Mamolo is in ArCo:

PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>

SELECT DISTINCT ?porta ?label
WHERE {
  ?porta rdf:type arco:ArchitecturalOrLandscapeHeritage .
  { ?porta rdfs:label ?label }
  UNION { ?porta arco-core:hasTitle ?title . ?title rdfs:label ?label }
  UNION { ?porta arco-core:hasDenomination ?den . ?den rdfs:label ?label }
  FILTER (REGEX(STR(?label), "Porta", "i"))
  FILTER (REGEX(STR(?label), "Bologna", "i"))
}
LIMIT 100

No results were found, confirming that Porta San Mamolo is not yet present in ArCo.

LIMIT 100

Explanation of relevant ArCo properties used

  • arco-cons:hasConstructionDevelopment / arco-cd:constructionPeriod: indicates the main construction period (XII century).
  • arco-den:hasMaterial: specifies the building material (brick).
  • arco-event:hasRelatedEvent: records the addition of the drawbridge in 1334.
  • arco-core:describes: stores descriptive information (e.g. dialect name).
  • arco-core:hasNote: records notes such as demolition date.

ASK and DESCRIBE queries

ASK query to check demolition

PREFIX arco-core: <https://w3id.org/arco/ontology/core/>

ASK {
  <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>
    arco-core:hasNote "Demolished in 1903"@en .
}

DESCRIBE query to retrieve all information

PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco-core: <https://w3id.org/arco/ontology/core/>

DESCRIBE <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>

The ASK returned false and the DESCRIBE returned empty, confirming the absence of Porta San Mamolo in the KG prior to insertion.

Comparison with LLMs

  • ChatGPT: helped generate CONSTRUCT queries and explanations, using a one-shot prompt and somewhat more generic URIs.
  • Gemini: provided structured queries with canonical ArCo URIs and readable RDF triples using zero-shot prompting.

Summary

Technical Summary: Semantic Enrichment of Porta San Mamolo using the ArCo ontology. The dataset includes construction period (12th century), Bolognese dialect name (Porta ed San Mamel), drawbridge addition (1334), demolition date (1903), and material (brick). SPARQL CONSTRUCT queries were prepared for ingestion into the ArCo KG. Verification queries (ASK / DESCRIBE) show the entity was absent before insertion.
