Porta San Mamolo - Semantic Enrichment with ArCo Ontology

Porta San Mamolo

Choice of the ‚Äúporta‚Äù and the information to be entered

Porta San Mamolo was one of Bologna‚Äôs city gates, historically significant for controlling the northern entrance. It was built in the 12th century and demolished in 1903. In 1334 a drawbridge was added. The Bolognese dialect name was Porta ed San Mamel. Its structure was made of bricks. I decided to enrich its profile using ArCo ontology given its historical and architectural importance.

Verification of the data present on ArCo

A SELECT query was executed to check for the presence of Porta San Mamolo in ArCo. No results were found, confirming that it is not yet present in the knowledge graph. 

PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>

SELECT DISTINCT ?porta ?label
WHERE {
  ?porta rdf:type arco:ArchitecturalOrLandscapeHeritage .
  { ?porta rdfs:label ?label }
  UNION { ?porta arco-core:hasTitle ?title .
          ?title rdfs:label ?label }
  UNION { ?porta arco-core:hasDenomination ?den .
          ?den rdfs:label ?label }
  FILTER (REGEX(STR(?label), "Porta", "i"))
  FILTER (REGEX(STR(?label), "Bologna", "i"))
}
LIMIT 100

Adding information with ChatGPT and ArCo properties

Using one-shot prompting, ChatGPT generated CONSTRUCT queries to model Porta San Mamolo within ArCo. 

# Example: Construction period
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco-cd: <https://w3id.org/arco/ontology/context-description/>

CONSTRUCT {
  <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>
    arco-cd:constructionPeriod "12th century"@en ;
    rdfs:label "Porta San Mamolo"@en .
}
WHERE {
  FILTER (true)
}

Other queries were created for building material (brick), the addition of the drawbridge in 1334, dialect name, and demolition date (1903).

Adding information with Gemini

Gemini was prompted in zero-shot mode, producing RDF triples directly using ArCo properties. The queries encoded construction period, dialect name, drawbridge addition, demolition, and building material. 

PREFIX arco-core: <https://w3id.org/arco/ontology/core/>
PREFIX arco-cons: <https://w3id.org/arco/ontology/construction-description/>
PREFIX arco-den: <https://w3id.org/arco/ontology/denotative-description/>

CONSTRUCT {
  <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>
    arco-cons:hasConstructionDevelopment "XII century"@en ;
    arco-core:describes "Bolognese dialect name: Porta ed San Mamel"@en .
}
WHERE { FILTER(true) }

ASK and DESCRIBE queries

Queries were used to check the presence of demolition information and retrieve all existing data. 

# ASK query
PREFIX arco-core: <https://w3id.org/arco/ontology/core/>
ASK {
  <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>
    arco-core:hasNote "Demolished in 1903"@en .
}

# DESCRIBE query
PREFIX arco-core: <https://w3id.org/arco/ontology/core/>
DESCRIBE <https://dati.beniculturali.it/resource/CulturalProperty/0800249999>

The ASK query returned false and the DESCRIBE query returned empty, confirming the absence of Porta San Mamolo in ArCo.

Comparison with LLMs

  • ChatGPT: generated CONSTRUCT queries with one-shot prompting, using more general URIs.
  • Gemini: produced structured RDF triples with precise ArCo URIs via zero-shot prompting.

Summary

Porta San Mamolo‚Äôs historical, architectural, and descriptive data were modeled in RDF using ArCo ontology. The enrichment includes construction period (12th century), dialect name (Porta ed San Mamel), addition of a drawbridge in 1334, demolition in 1903, and building material (brick). Both ChatGPT and Gemini assisted in generating SPARQL queries, showing complementary strengths.
