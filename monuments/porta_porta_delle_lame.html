Porta delle Lame (also known as Porta Lame)

Porta delle Lame is one of the historic gates of Bologna, dating back to the medieval walls. It is also referred to as Porta Lame.It could also be called the "Gate of the Waters." This is not only because of its proximity to the port and the nearby salt warehouse but also because the road leading from it went to the many bodies of water that covered the low-lying plain. In Latin, lama means pond or swamp. Built in the 13th century with a covered fortress, it was equipped with two drawbridges in 1334‚Äîone for carriages and one for pedestrians. The gate was closed several times because it was difficult to garrison. Between 1674 and 1677, it was radically renovated, with the demolition of the medieval fortress and the construction of the current Baroque-style building. Among the events in which the gate played a leading role was a recent one: the Battle of November 1944, when, after an alternating series of phases, the partisans prevailed over a contingent of Nazi-fascist troops. This page presents the results of SPARQL queries executed on the ArCo dataset to retrieve structured information about this monument.

SPARQL Query 1: Retrieve the Gate

The following query searches for architectural heritage labeled as Porta delle Lame in Bologna: 

PREFIX rdf:  <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX arco-core: <https://w3id.org/arco/ontology/core/>
PREFIX cis:  <http://dati.beniculturali.it/cis/>

SELECT DISTINCT ?porta ?label
WHERE {
  ?porta rdf:type arco:ArchitecturalOrLandscapeHeritage .
  { ?porta rdfs:label ?label }
  UNION { ?porta cis:name ?label }
  UNION { ?porta arco-core:hasTitle ?title .
           ?title rdfs:label ?label }
  UNION { ?porta arco-core:hasDenomination ?den .
           ?den rdfs:label ?label }
  FILTER (REGEX(STR(?label), "Porta Lame", "i"))
  FILTER (REGEX(STR(?label), "Bologna", "i"))
}
LIMIT 100

SPARQL Query 9: SELECT

SELECT query about Porta Lame to find out if some properties exist in ArCo 

 
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX a-cd: <https://w3id.org/arco/ontology/context-description/>
PREFIX a-dd: <https://w3id.org/arco/ontology/denotative-description/>
PREFIX a-arch: <https://w3id.org/arco/ontology/architectural-description/>

SELECT DISTINCT ?culturalProperty ?title
       (BOUND(?architecturalOrder) AS ?hasArchitecturalOrder)
       (BOUND(?constructionDevelopment) AS ?hasConstructionDevelopment)
       (BOUND(?constructionElement) AS ?hasConstructionElement)
       (BOUND(?constructionSpace) AS ?hasConstructionSpace)
       (BOUND(?covering) AS ?hasCovering)
WHERE {
  {
    VALUES ?culturalProperty { <https://w3id.org/arco/resource/ArchitecturalOrLandscapeHeritage/0800241649> }

    OPTIONAL { ?culturalProperty rdfs:label ?title . }
    OPTIONAL { ?culturalProperty a-arch:hasArchitecturalOrder ?orderNode . ?orderNode rdfs:label ?architecturalOrder . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionDevelopment ?devNode . ?devNode rdfs:label ?constructionDevelopment . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionElement ?elemNode . ?elemNode rdfs:label ?constructionElement . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionSpace ?spaceNode . ?spaceNode rdfs:label ?constructionSpace . }
    OPTIONAL { ?culturalProperty a-arch:hasCovering ?covNode . ?covNode rdfs:label ?covering . }
  }
  UNION
  {
    VALUES ?culturalProperty { <https://w3id.org/arco/resource/ArchitecturalOrLandscapeHeritage/0800242005> }

    OPTIONAL { ?culturalProperty rdfs:label ?title . }
    OPTIONAL { ?culturalProperty a-arch:hasArchitecturalOrder ?orderNode . ?orderNode rdfs:label ?architecturalOrder . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionDevelopment ?devNode . ?devNode rdfs:label ?constructionDevelopment . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionElement ?elemNode . ?elemNode rdfs:label ?constructionElement . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionSpace ?spaceNode . ?spaceNode rdfs:label ?constructionSpace . }
    OPTIONAL { ?culturalProperty a-arch:hasCovering ?covNode . ?covNode rdfs:label ?covering . }
  }
    FILTER(REGEX(?title, "Porta", "i"))
}
ORDER BY ?title
LIMIT 10
