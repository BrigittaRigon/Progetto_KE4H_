Porta delle Lame (also known as Porta Lame)

Porta delle Lame is one of the historic gates of Bologna, dating back to the medieval walls. It is also referred to as Porta Lame. This page presents the results of SPARQL queries executed on the ArCo dataset to retrieve structured information about this monument.

SPARQL Query 1: Retrieve the Gate

The following query searches for architectural heritage labeled as Porta delle Lame in Bologna: 

PREFIX rdf: 
PREFIX rdfs: 
PREFIX arco: 
PREFIX a-cd: 
PREFIX a-dd: 
PREFIX a-arch: 

SELECT DISTINCT ?culturalProperty ?title
       (BOUND(?architecturalOrder) AS ?hasArchitecturalOrder)
       (BOUND(?constructionDevelopment) AS ?hasConstructionDevelopment)
       (BOUND(?constructionElement) AS ?hasConstructionElement)
       (BOUND(?constructionSpace) AS ?hasConstructionSpace)
       (BOUND(?covering) AS ?hasCovering)
WHERE {
  {
    VALUES ?culturalProperty {  }

    OPTIONAL { ?culturalProperty rdfs:label ?title . }
    OPTIONAL { ?culturalProperty a-arch:hasArchitecturalOrder ?orderNode . ?orderNode rdfs:label ?architecturalOrder . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionDevelopment ?devNode . ?devNode rdfs:label ?constructionDevelopment . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionElement ?elemNode . ?elemNode rdfs:label ?constructionElement . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionSpace ?spaceNode . ?spaceNode rdfs:label ?constructionSpace . }
    OPTIONAL { ?culturalProperty a-arch:hasCovering ?covNode . ?covNode rdfs:label ?covering . }
  }
  UNION
  {
    VALUES ?culturalProperty {  }

    OPTIONAL { ?culturalProperty rdfs:label ?title . }
    OPTIONAL { ?culturalProperty a-arch:hasArchitecturalOrder ?orderNode . ?orderNode rdfs:label ?architecturalOrder . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionDevelopment ?devNode . ?devNode rdfs:label ?constructionDevelopment . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionElement ?elemNode . ?elemNode rdfs:label ?constructionElement . }
    OPTIONAL { ?culturalProperty a-arch:hasConstructionSpace ?spaceNode . ?spaceNode rdfs:label ?constructionSpace . }
    OPTIONAL { ?culturalProperty a-arch:hasCovering ?covNode . ?covNode rdfs:label ?covering . }
  }
    FILTER(REGEX(?title, "Porta", "i"))
}
ORDER BY ?title
LIMIT 10

Discussion

The queries confirm the existence of Porta delle Lame in the ArCo knowledge graph, associated with Bologna and classified as architectural heritage. The DESCRIBE query provides a rich set of metadata, including identifiers, classifications, and potential references to restoration events.

Conclusion

Porta delle Lame (Porta Lame) illustrates how Linked Open Data and SPARQL queries can be applied to retrieve and analyze cultural heritage information. This structured approach supports digital documentation, visualization, and comparative studies with other historic gates of Bologna.
